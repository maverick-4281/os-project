# Secure File Management System

A Python + Flask based Secure File Management System for the CSE-316 Operating Systems subject project.

## Overview

This project is designed to provide secure file handling with a modular architecture and OS-security focused concepts.

## Planned Core Features

- Authentication with username/password and 2FA (OTP)
- File operations: read, write, encrypt, share, and metadata management
- Threat detection: buffer overflow checks and malware scanning

## Tech Stack

- Python
- Flask
- Security libraries such as bcrypt, pyotp, cryptography, and qrcode

## Academic Context

Developed as part of the CSE-316 OS subject work.
